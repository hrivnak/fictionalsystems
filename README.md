## 💰 Pricing & Operational Model

DocRobot is designed for **total autonomy**. Unlike cloud-based solutions, there are no mandatory monthly fees. You own the hardware and the intelligence.

### 1. Initial Investment (One-time)
* **Turnkey Setup:** 1,600 – 3,200 EUR  
  *(Includes: Hardware procurement, OS optimization, Local LLM integration, and initial RAG database configuration).*

---

### 2. Service & Support Structure

| Service Tier | Cost | Description |
| :--- | :--- | :--- |
| **Daily Maintenance** | **0 EUR / month** | The system runs 100% locally on your infrastructure. No subscriptions. |
| **Security Patches** | **Included** | Core system and security updates for the first 12 months. |
| **Model Migrations** | **On-Demand** | One-time fee for upgrading to future LLM architectures (e.g., Llama 4/5). |
| **Expert Support** | **Custom SLA** | Priority technical assistance and advanced document indexing services. |

---

### 🛡️ Why 0 EUR Maintenance?
Our philosophy is built on **Data Sovereignty**. Once the DocRobot workstation is inside your network:
* **No "Lock-in":** You are not dependent on external API availability.
* **Predictable Costs:** Fixed CAPEX (Initial investment) vs. unpredictable OPEX (Monthly fees).
* **Scaling for Free:** Adding more internal users does not increase your monthly costs.
